# Snake (written in AWK)

This project is an exercise in making things work even when they probably shouldn't.

## Requirements*

- GNU Awk (v5.3.1)
- Bash (v5.2.37)

*These are the versions I used while testing. Support for others may vary (especially if you aren't using GNU Awk).*

## Playing the Game

The entire game is contained within `snake.awk`. Assuming you have AWK you should be able to run the game as shown below:

```shell
./snake.awk
```
